逢甲幹課 Beta
===========================


## 環境需求
* python 3.6.3 up
* selenium
* chromedriver
	* https://sites.google.com/a/chromium.org/chromedriver/downloads
* chrome正式發行版本
* 一顆懷抱著感恩的心

## 如何使用？
#### 輸入帳號密碼
	username = ''
    password = ''
#### 輸入想要幹的選課代碼
	classID = '1234'
#### 搶課Flag設為True
	grabbed = True
#### 執行並等待有人不選這堂課
#### 祝你好運！
